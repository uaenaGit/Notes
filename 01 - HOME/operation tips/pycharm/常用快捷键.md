### 🚀 1. 最核心的“救命”快捷键 (必背)
| 功能         | Windows / Linux        | Mac                      | 说明                                           |
| :--------- | :--------------------- | :----------------------- | :------------------------------------------- |
| **命令面板**   | `Ctrl` + `Shift` + `P` | `Cmd` + `Shift` + `P`    | 万能键。找不到功能时按它，输入关键词（如 "Format", "Theme"）即可执行。 |
| **快速打开文件** | `Ctrl` + `P`           | `Cmd` + `P`              | 模糊搜索文件名，快速跳转。                                |
| 智能提示/补全    | `Ctrl` + `Space`       | `Ctrl` + `Space`         | 触发代码自动补全（如果没弹出来就按这个）。                        |
| 格式化文档      | `Shift` + `Alt` + `F`  | `Shift` + `Option` + `F` | 一键美化代码缩进和格式。                                 |
| 注释/取消注释    | `Ctrl` + `/`           | `Cmd` + `/`              | 快速注释当前行或选中行（切换 `//` 或 `#`）。                  |
| 显示/隐藏侧边栏   | `Ctrl` + `B`           | `Cmd` + `B`              | 最大化编辑区域。                                     |

### ✍️ 2. 编辑与多光标神技 (提升手速)
这是 VS Code 最强大的地方，学会多光标编辑，效率翻倍。

| 功能           | Windows / Linux         | Mac                        | 说明                                          |
| :----------- | :---------------------- | :------------------------- | :------------------------------------------ |
| **向上/下复制行**  | `Shift` + `Alt` + `↓/↑` | `Shift` + `Option` + `↓/↑` | 快速复制当前行到下方或上方。                              |
| **向上/下移动行**  | `Alt` + `↓/↑`           | `Option` + `↓/↑`           | 不复制，直接交换当前行位置。                              |
| **删除当前行**    | `Ctrl` + `Shift` + `K`  | `Cmd` + `Shift` + `K`      | 直接删掉整行，无需选中。                                |
| **添加下一个匹配项** | `Ctrl` + `D`            | `Cmd` + `D`                | **神器**。选中一个词，再按一次会选中下一个相同的词，可连续按，实现多光标同时编辑。 |
| **选中所有匹配项**  | `Ctrl` + `Shift` + `L`  | `Cmd` + `Shift` + `L`      | 一次性选中文件中所有相同的词，全部一起改。                       |
| 多光标点击        | `Alt` + `Click`         | `Option` + `Click`         | 鼠标按住 Alt 点击多处，生成多个光标。                       |
| 列选择模式        | `Shift` + `Alt` + `拖拽`  | `Shift` + `Option` + `拖拽`  | 垂直框选一块区域进行编辑。                               |
| 行尾插入光标       | `Shift` + `Alt` + `I`   | `Shift` + `Option` + `I`   | 选中多行后按此键，会在每一行的末尾都加上一个光标（适合批量加分号或逗号）。       |

### 🔍 3. 查找与替换
| 功能          | Windows / Linux        | Mac                    | 说明              |
| :---------- | :--------------------- | :--------------------- | :-------------- |
| **当前文件查找**  | `Ctrl` + `F`           | `Cmd` + `F`            | 基础查找。           |
| 当前文件替换      | `Ctrl` + `H`           | `Cmd` + `Option` + `F` | 基础替换。           |
| **全局查找**    | `Ctrl` + `Shift` + `F` | `Cmd` + `Shift` + `F`  | 在整个项目所有文件中查找。   |
| 全局替换        | `Ctrl` + `Shift` + `H` | `Cmd` + `Shift` + `H`  | 在整个项目所有文件中替换。   |
| **跳转到定义**   | `F12`                  | `F12`                  | 跳到函数或变量的定义处。    |
| **查看引用**    | `Shift` + `F12`        | `Shift` + `F12`        | 查看哪里使用了这个变量/函数。 |
| **回到上一个位置** | `Alt` + `←`            | `Ctrl` + `-`           | 跳完定义后，用这个键跳回来。  |

| 切换焦点        | `Ctrl` + `1` / `2`         | `Cmd` + `1` / `2`         | 在左右分屏间切换光标焦点。 |
| **打开/关闭终端** | `Ctrl` + `` ` `` (反引号)     | `Ctrl` + `` ` ``          | 快速唤出底部终端。     |
| 新建终端        | `Ctrl` + `Shift` + `` ` `` | `Cmd` + `Shift` + `` ` `` | 开一个新的终端窗口。    |
| 关闭当前编辑器     | `Ctrl` + `W`               | `Cmd` + `W`               | 关闭当前打开的文件标签页。 |
