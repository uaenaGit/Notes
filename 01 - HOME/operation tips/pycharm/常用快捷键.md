### 🚀 1. 最核心的“救命”快捷键 (必背)
这些是每天使用频率最高的键，掌握它们能解决 80% 的操作需求。

| 功能                           | Windows / Linux            | Mac                       | 说明                                         |
| :--------------------------- | :------------------------- | :------------------------ | :----------------------------------------- |
| **万能搜索** (Search Everywhere) | `Shift` + `Shift` (连按两次)   | `Shift` + `Shift`         | 最强功能。搜文件、类、符号、设置、甚至菜单项。找不到东西就按它！           |
| 运行当前文件                       | `Shift` + `F10`            | `Ctrl` + `R`              | 直接运行当前编辑的 Python 脚本。                       |
| 调试当前文件                       | `Shift` + `F9`             | `Ctrl` + `D`              | 启动 Debug 模式。                               |
| 智能代码补全                       | `Ctrl` + `Space`           | `Ctrl` + `Space`          | 基础补全。如果没反应，检查是否被输入法占用。                     |
| 智能类型补全                       | `Ctrl` + `Shift` + `Space` | `Cmd` + `Shift` + `Space` | 更聪明。根据上下文变量类型推荐属性和方法（比基础补全更精准）。            |
| 快速修复/优化导入                    | `Alt` + `Enter`            | `Option` + `Enter`        | 红灯泡神器。自动导包、修复语法错误、创建函数、转换格式等。              |
| **重命名 (重构)**                 | `Shift` + `F6`             | `Shift` + `F6`            | 安全改名。修改变量/函数名时，会自动更新所有引用该名字的地方（比全局替换安全得多）。 |
| 格式化代码                        | `Ctrl` + `Alt` + `L`       | `Cmd` + `Option` + `L`    | 一键美化代码缩进和空格（PEP 8 标准）。                     |
| 显示参数提示                       | `Ctrl` + `P`               | `Cmd` + `P`               | 在函数括号内时，显示该函数需要哪些参数。                       |
| 查看定义                         | `Ctrl` + `B` (或鼠标中键点击)     | `Cmd` + `B`               | 跳转到函数或类的定义处。                               |
| 查看用法                         | `Alt` + `F7`               | `Option` + `F7`           | 查看这个变量/函数在哪里被使用了。                          |
| 最近的文件                        | `Ctrl` + `E`               | `Cmd` + `E`               | 快速切换最近打开过的文件。                              |
| 注释/取消注释                      | `Ctrl` + `/`               | `Cmd` + `/`               | 快速切换 `#` 注释。                               |

### ✍️ 2. 编辑与多光标 (提升手速)

PyCharm 的多光标功能同样强大，但部分快捷键与 VS Code 不同。

| 功能          | Windows / Linux                | Mac                         | 说明                                                |
| :---------- | :----------------------------- | :-------------------------- | :------------------------------------------------ |
| **复制当前行**   | `Ctrl` + `D`                   | `Cmd` + `D`                 | 注意：这里不是选中下一个，而是复制整行（同 VS Code 的 `Shift+Alt+↓`）。   |
| **删除当前行**   | `Ctrl` + `Y`                   | `Cmd` + `Delete`            | 直接删掉整行。                                           |
| 上下移动行       | `Alt` + `Shift` + `↑/↓`        | `Option` + `Shift` + `↑/↓`  | 交换当前行与上/下行的位置。                                    |
| 添加下一个匹配项    | `Alt` + `J`                    | `Ctrl` + `G`                | 对应 VS Code 的 `Ctrl+D`。选中一个词，按此键选中下一个相同的词，实现多光标编辑。 |
| **选中所有匹配项** | `Ctrl` + `Alt` + `Shift` + `J` | `Ctrl` + `Cmd` + `G`        | 对应 VS Code 的 `Ctrl+Shift+L`。一次性选中所有相同词。           |
| 多光标点击       | `Alt` + `Click`                | `Option` + `Click`          | 鼠标按住 Alt 点击多处生成光标。                                |
| 列选择模式       | `Alt` + `Shift` + `拖拽`         | `Option` + `Shift` + `拖拽`   | 垂直框选编辑。                                           |
| 展开/折叠代码块    | `Ctrl` + `+` / `-`             | `Cmd` + `+` / `-`           | 折叠函数或类，让代码更简洁。                                    |
| 展开/折叠所有     | `Ctrl` + `Shift` + `+` / `-`   | `Cmd` + `Shift` + `+` / `-` | 全部展开或全部折叠。                                        |

### 🔍 3. 查找与替换
| 功能           | Windows / Linux                | Mac                    | 说明                                        |
| :----------- | :----------------------------- | :--------------------- | :---------------------------------------- |
| **当前文件查找**   | `Ctrl` + `F`                   | `Cmd` + `F`            | 基础查找。                                     |
| 当前文件替换       | `Ctrl` + `R`                   | `Cmd` + `R`            | 注意：在 PyCharm 中这是替换，不是运行（运行是 `Shift+F10`）。 |
| **全局查找**     | `Ctrl` + `Shift` + `F`         | `Cmd` + `Shift` + `F`  | 在整个项目中查找。                                 |
| 全局替换         | `Ctrl` + `Shift` + `R`         | `Cmd` + `Shift` + `R`  | 在整个项目中替换。                                 |
| 查找文件路径       | `Ctrl` + `Shift` + `N`         | `Cmd` + `Shift` + `O`  | 专门通过文件名查找文件（比 `Shift+Shift` 更专注）。         |
| 查找类          | `Ctrl` + `N`                   | `Cmd` + `O`            | 专门通过类名查找类定义。                              |
| 查找符号 (函数/变量) | `Ctrl` + `Alt` + `Shift` + `N` | `Cmd` + `Option` + `O` | 通过函数名或变量名直接定位代码。                          |

### 🛠️ 4. 调试专用 (Debug)
当程序进入断点模式后，这些键至关重要：

| 功能                     | Windows / Linux         | Mac                       | 说明                                |
| :--------------------- | :---------------------- | :------------------------ | :-------------------------------- |
| **单步跳过** (Step Over)   | `F8`                    | `F8`                      | 执行当前行，如果行内有函数，不进入函数内部，直接执行完。      |
| **单步进入** (Step Into)   | `F7`                    | `F7`                      | 如果当前行有函数，进入函数内部逐行调试。              |
| 强制进入 (Force Step Into) | `Alt` + `Shift` + `F7`  | `Option` + `Shift` + `F7` | 即使进入库函数（如 `print`）内部也强行进入（通常用不到）。 |
| 跳出函数 (Step Out)        | `Shift` + `F8`          | `Shift` + `F8`            | 执行完当前函数剩余部分，返回到调用处。               |
| 恢复程序 (Resume)          | `F9`                    | `Cmd` + `Alt` + `R`       | 继续运行直到下一个断点。                      |
| 切换断点                   | `Ctrl` + `F8`           | `Cmd` + `F8`              | 在当前行打点或取消断点。                      |
| 查看所有断点                 | `Ctrl` + `Shift` + `F8` | `Cmd` + `Shift` + `F8`    | 弹出断点管理窗口。                         |

### 总结：新手最先练熟这 5 个

1. `Shift` + `Shift` (找不到东西就按它)
2. `Alt` + `Enter` (报红了按它，想导包按它)
3. `Shift` + `F6` (改名专用，安全无忧)
4. `Ctrl` + `/` (注释代码)
5. `Ctrl` + `Alt` + `L` (代码太乱，一键美化)
