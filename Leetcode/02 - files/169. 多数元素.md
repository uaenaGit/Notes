> 🔗[169 - 多数元素](https://leetcode.cn/problems/majority-element/description/?envType=study-plan-v2&envId=top-interview-150)

## 📌 题目描述
> （请在此处粘贴题目描述）

给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

**示例 1：**

**输入：** nums = [3,2,3]
**输出：**3

**示例 2：**

**输入：** nums = [2,2,1,1,1,2,2]
**输出：** 2

**提示：**

- `n == nums.length`
- $1 <= n <= 5 * 10^4$
- $-10^9 <= nums[i] <= 10^9$
- 输入保证数组中一定有一个多数元素。

## 💡 解题思路 
利用摩尔投票法：记众数为 `x` ，票数统计 `votes = 0`。
遍历数组`nums`，
- 若 `nums[i] == x` ：将票数`votes`增 $1$。
- 若 `nums[i] != x` :  将票数`votes`减 $1$。
注意当`votes == 0`时，更新待选众数为`nums[i]`，同时将 `votes` 增 $1$。
## 💻 代码实现
> my answer
```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int vote = 0;
        int majority = INT_MAX;
        for (auto &num : nums) {
            if (vote == 0) {
                majority = num; // 选最新的元素
                vote += 1; // vote 增 1
            } else if (majority == num) {
                vote += 1; 
            } else {
                vote -= 1;
            }
        }
        return majority;

    }
};
```

- **时间复杂度**：`O(n)`
- **空间复杂度**：`O(1)`

> to optimize

将`nums[i]` 与 众数 `x`的比较以及票数`votes`的更新使用一行代码
```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int votes = 0;
        int majority = INT_MAX;
        for (auto &num : nums) {
            if (votes == 0) {
                majority = num; // 选新的元素成为众数
            }
            votes += (majority == num) ? 1 : -1;
        }
        return majority;
    }
};
```
- **时间复杂度**：`O(n)`
- **空间复杂度**：`O(1)`
