> 🔗[121 - 买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/solutions/2464650/mei-ju-mai-chu-jie-ge-wei-hu-mai-ru-de-z-02ud/?envType=study-plan-v2&envId=top-interview-150)

## 📌 题目描述
> （请在此处粘贴题目描述）

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

**示例 1：**

**输入：**[7,1,5,3,6,4]
**输出：**5
**解释：**在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

**示例 2：**

**输入：**prices = [7,6,4,3,1]
**输出：**0
**解释：**在这种情况下, 没有交易完成, 所以最大利润为 0。

**提示：**

- $1 <= prices.length <= 10^5$
- $0 <= prices[i] <= 10^4$

## 💡 解题思路 

从后往前遍历：维护最大值，更新最大利润。
从前往后遍历：维护最小值，更新最大利润。
## 💻 代码实现
> my answer
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int rightMax = 0;
        int dif = 0;
        // 维护右边的最大值，更新最大利润
        for (int i = prices.size() - 1; i >= 0; i--) {
            dif = max(dif, rightMax - prices[i]);
            rightMax = max(rightMax, prices[i]);
        }
        return dif;
    }
}; 
```
- **时间复杂度**：`O(n)`。$n$ 为数组长度。
- **空间复杂度**：`O(1)`

> to optimize

从前往后遍历，维护最小值，更新最大利润
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int cost = INT_MAX;
        int ans = 0;
        for (auto &price : prices) {
            ans = max(ans, price - cost);
            cost = min(cost, price);
        }
        return ans;
    }
}; 
```
- **时间复杂度**：`O(n)`。$n$ 是数组长度
- **空间复杂度**：`O(1)`

>✨ tips
- `INT_MAX`在 `cpp`的`<climits>`库中。
